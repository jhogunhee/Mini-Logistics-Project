🚀 프로젝트 소개

본 프로젝트는 React + Vite 환경에서
실제 물류 시스템(WMS/OMS) 프로젝트를 수행하며 경험했던
기준정보(MDM) 설계, 재고 관리, 입·출고 흐름을 다시 정리하고
프론트엔드 구조 설계 역량을 강화하기 위해 진행하는 개인 프로젝트입니다.

단순 CRUD 구현이 아닌,
실무에서 사용되는 화면 구조와 데이터 흐름을 최대한 반영하는 것을 목표로 합니다.
